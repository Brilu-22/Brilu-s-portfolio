MOVE FitCal is an AI-powered fitness companion where sports performance meets iOS elegance. This intelligent platform transforms fitness journeys through personalized AI guidance, combining cutting-edge technology with premium design to create the ultimate fitness experience.

The Vision

"We believe fitness technology should be as beautiful as it is functional - inspiring movement through exceptional design." MOVE was built to bridge the gap between complex fitness tracking and intuitive user experience, leveraging AI to provide truly personalized fitness and nutrition guidance.

Real user testing revealed exceptional satisfaction across all metrics. Users praised the intuitive interface, smooth navigation, and the AI's ability to understand their fitness needs.

Core Features

AI Workout Plans

Personalized routines using Google Gemini 2.5 Pro, adaptive duration (30-90 mins), body part focused training with progressive overload principles.

Smart Nutrition

AI-generated meal recommendations with Edamam nutrition analysis, ingredient tracking, and dietary preference customization.

Progress Tracking

Comprehensive weight trend visualization, fitness metrics dashboard, goal achievement tracking with weekly analytics.

Beautiful UI

iOS-inspired design language with smooth animations, system-wide dark mode, and WCAG 2.1 AA accessibility compliance.

Technical Excellence

Modern Tech Stack

React Native 0.81.5 TypeScript 5.9 Expo Router 6.0 ASP.NET Core 8.0 Entity Framework Google Gemini 2.5 Pro Edamam API Firebase Auth

The architecture features a React Native frontend with Expo for cross-platform development, backed by a robust ASP.NET Core API. AI integration through Google Gemini provides intelligent workout generation, while Edamam powers comprehensive nutrition analysis.

Performance Metrics

Design Philosophy

Athletic Energy

Dynamic color gradients inspired by sports apparel, motion design reflecting physical activity, energetic yet professional tone.

iOS Clarity

SF Pro typography system, consistent 8pt grid spacing, familiar navigation patterns for instant usability.

Dark Theme Excellence

True black backgrounds for OLED optimization, carefully calibrated contrast ratios, reduced eye strain during extended use.

Accessibility First

WCAG 2.1 AA compliance throughout, VoiceOver and TalkBack support, dynamic type scaling.
